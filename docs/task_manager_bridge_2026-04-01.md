# 任务管理桥接记录

## 目标

把 AngelHeart 对 AstrBot future task 的桥接，从“只会新增提醒任务”扩到“能查、能删、能明确并存”的可管理状态。

## 本轮范围

- 支持列出当前会话下的 future task
- 支持按关键词删除当前会话下的 future task
- 支持在新增任务确认时明确说明“不会覆盖旧任务”

## 风险边界

“覆盖旧任务”如果没有明确旧任务标识，单靠自然语言很容易误删。因此本轮不做激进的自动覆盖，只提供：

- 明确并存
- 明确列出
- 明确删除

后续如果要做“改成/覆盖旧任务”，建议优先结合：

- 引用的旧任务消息
- job_id
- 或先列出再让用户确认删除目标
